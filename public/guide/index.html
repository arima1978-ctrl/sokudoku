ステージ別トレーニング内容

1

Stage 1: 3点読み

最低6回 / 4種目を順番に

ばらばら → 1行読み → 2行読み → ブロック読み → 高速読み
方向: 1回目たて → 2回目よこ → 3回目たて → ...（奇数回=たて、偶数回=よこ）
2

Stage 2: 2点読み

最低8回 / 1行と2行を交互

ばらばら → 1行 or 2行 → ブロック読み → 高速読み
交互ルール: 1,2回目→1行読み / 3,4回目→2行読み / 5,6回目→1行読み / 7,8回目→2行読み（4回周期）
3

Stage 3: 1行読み

最低8回 / ばらばら省略

1行読み → 2行読み → ブロック読み → 高速読み
4

Stage 4: 2行読み

最低8回 / 視点移動が時間の半分

ばらばら → ブロック読み → 視点移動 (約50%) → 高速読み
5

Stage 5: ブロック読み

最低8回 / 高速処理能力の完成

ブロック読み → 視点移動 (約50%) → 高速読み
∞

スピードモード

Stage 5 完了後 / カウント青天井

ブロック読み (50%) → 視点移動 (50%) → 高速読み
どこまでもカウントを上げ続けるトレーニング。上限なし。
