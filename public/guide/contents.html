📥 コンテンツパックを一括インポート

Excel / CSV でまとめて登録できます
📋 STEP 1: テンプレートをダウンロード
まずはテンプレートをダウンロードして、指定フォーマットに沿って記入してください。
📝 テンプレート構造（4シート）
シート名 内容 行数目安
📘 Packs タイトル/ジャンル/対象コース/声絵/学年/難易度 1行＝1パック
① Barabara pack_id / 単語 50+/パック
② Line1 pack_id / 1行文 50+/パック
③ Line2 pack_id / 1行目 / 2行目 50+/パック
④ Main pack_id / 高速読み本文 1行＝1パック
🎯 Quizzes pack_id / 問題文 / A / B / C / D / 正解 5問/パック
📤 STEP 2: ファイルをアップロード
📁
ファイルをドラッグ&ドロップ または
対応形式: .xlsx, .xls, .csv ／ 最大10MB
✓ STEP 3: プレビューと確認
アップロード後、以下を確認してからインポート実行します:
  • パック数 / 総問題数 / エラー行
  • 文字数オーバー・必須項目欠落の検出
  • 既存パックとの重複検出（タイトルで判定）
  • インポート方法: 新規追加のみ / 上書き更新 / スキップ
⚠️ 一括インポート時は、pack_id を各シートで共通にしてパックとテキストを紐付けます。テンプレートの記入例を参考にしてください。
